Tabla (entidad).

- Definición: Una tabla es una colección de datos organizados en un formato estructurado, que se compone de filas y columnas. Cada tabla representa una entidad específica en el modelo de datos.

- Ejemplo: En una base de datos de una biblioteca, podrías tener una tabla llamada "Libros" que almacena información sobre todos los libros disponibles.

Registro .

- Definición: Un registro es una fila en una tabla que contiene todos los datos relacionados con una única instancia de la entidad que representa la tabla. Cada registro es único y se identifica generalmente por una clave primaria.

- Ejemplo: En la tabla "Libros", un registro podría contener información sobre un libro específico, como su título, autor, año de publicación, etc. Por ejemplo:
- Registro: (1, "Cien años de soledad", "Gabriel García Márquez", 1967)

Campo (Atributo).

- Definición: Un campo es una columna en una tabla que representa un atributo específico de la entidad. Cada campo tiene un nombre y un tipo de dato asociado (como texto, número, fecha, etc.). - Ejemplo: En la tabla "Libros", podrías tener varios campos como: - ID (número entero) - Título (texto) - Autor (texto) - Año de Publicación (número entero)
